# Description : 
A to-do list is a list of things that need to be completed. Items on a to-do list can be anything, from simple tasks like replying to an email to more complex tasks like creating project briefs. To-do lists can help you stay on top of your tasks and projects, avoid missing deadlines, and remember important tasks.

## Features : 
### 1. Edit/Delete functionality
### 2. Voice Recognition 
### 3. Clear in one tap
